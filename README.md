# تطبيق خزانة

تطبيق لأجهزة الحاسوب (وينداوز، ماك، لينيكس) لتحميل المقاطع والدروس الصوتية من اليوتيوب

## شروط استعمال التطبيق

الشرط الوحيد هو عدم استعماله في المحرمات، كالموسيقى والأنمي والأفلام وغيرها مما لا يرضى الله.
